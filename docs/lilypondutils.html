LilypondUtils¶

class musicworkbook.LilypondUtils[source]¶

Gradually builds up content for a Lilypond file. When the content is complete, this class can create a Lilypond file.

buildLilypondKeyBlock(keyData)[source]¶

From the given key data, build a Lilypond /score block for that key. That block represents scale and arpeggio info for a specific key
