getNextNoteDown(targetNote, pivotNote)[source]¶

Starting from a pivot note, gets the next instance of the target note below the pivot note. For example, if the pivot note is C4 and target note is G, this function would return G3.

getNextNoteUp(targetNote, pivotNote)[source]¶

Starting from a pivot note, gets the next instance of the target note above the pivot note. For example, if the pivot nNote is C4 and target note is G, this function would return G4.
