LilypondFileBuilder¶

class musicworkbook.LilypondFileBuilder(clef=None, instrument=None, filename=None)[source]¶

Builds up étude content for a Lilypond file, using data from the WorkbookBuilder. When the content is ready, this class can create a Lilypond file with that content.
